Dashboard Mercado Financiero – Análisis Técnico

Este archivo resume el proyecto de análisis técnico del mercado financiero argentino. Incluye un conjunto de datos ficticio para ilustrar la estructura del tablero.

Descripción

El dashboard interactivo integra indicadores clave de precios y volúmenes, permitiendo evaluar tendencias y patrones mediante gráficos dinámicos y tablas pivotantes. Se utilizan técnicas de análisis técnico como medias móviles, bandas de Bollinger y osciladores de momentum para identificar oportunidades de inversión.
